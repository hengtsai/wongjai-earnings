## 2. 🔍 隱藏的魔鬼細節

<div class="sb">
<h3>🎯 洞察一：毛利率下滑 5.7pts 的結構性原因</h3>
<p>Q3 FY26 毛利率 64.6% vs Q3 FY25 的 70.3%，<b>看似大幅惡化，實際是業務組合的結構性轉變</b>。OCI（IaaS）營收 YoY +84%，而 IaaS 毛利率天然低於 SaaS。這是「成長的代價」：低毛利的基礎設施業務快速成長拉低整體毛利率，但營業利益率反而從 31.3% 升至 32.7%，因為營收規模效應消化了毛利壓力。</p>
</ul>
</div>

<div class="sb">
<h3>🎯 洞察二：資本支出暴增 → FCF 大幅轉負</h3>
<p>TTM CapEx <b>$482.5 億</b> vs FY25 的 $212.2 億（YoY +127%），導致 TTM 自由現金流轉為 <b>-$247.4 億</b>。這不是經營問題，而是 Oracle 正處於「基礎設施建設超級週期」。管理層表示：</p>
<ul>
<li>已取得 <b>10GW+</b> 電力與資料中心容量</li>
<li>超過 90% 容量由合作夥伴出資，其餘本月完工</li>
<li>2026 年計劃融資 $500 億，已到位 $300 億（投資級債 + 強制可轉換優先股）</li>
<li>CapEx 的超額支出實際上由外部融資支撐，非消耗自有現金</li>
</ul>
</div>

<div class="sb">
<h3>🎯 洞察三：RPO $5,530 億 = 8.6 倍年營收的「安全墊」</h3>
<p>RPO（剩餘履約義務）$5,530 億，<b>YoY +325%</b>，單季增加 $290 億。這個數字約為 TTM 營收的 8.6 倍，創下 Oracle 歷史紀錄。這意味著即使未來完全沒有新合約，Oracle 仍有 8 年以上的營收可認列。主要來自大型 AI 基礎設施合約（Stargate 等 multi-year 訂單）。<b>風險</b>：大型合約履約速度取決於資料中心完工進度。</p>
</div>

<div class="sb">
<h3>🎯 洞察四：MultiCloud DB +531% — 被低估的成長催化劑</h3>
<p>Oracle 資料庫已在 AWS、Azure、GCP 全球所有區域上線。MultiCloud DB 營收 YoY +531%，雖然基期小，但這代表 Oracle 打破了「客戶必須使用 OCI」的限制，將最強產品（自治資料庫）直接送進競爭對手的雲端。這個策略可能重塑資料庫市場格局，因為企業可以在不遷移雲端的情況下使用 Oracle DB。</p>
</div>

<div class="sb">
<h3>🎯 洞察五：Q3 Non-GAAP EPS 超預期但「差額」不大</h3>
<p>Non-GAAP EPS $1.79 vs 預期 $1.70，超出 <b>$0.09（+5.3%）</b>。超幅溫和，不像過去幾季的大幅 beat。市場可能已經將更高的成長期望 price-in 到股價中。</p>
</div>

<div class="sb">
<h3>🎯 洞察六：TikTok US 15% 股權 — 新的非營業收入來源</h3>
<p>Oracle 於 2026 年 1 月取得 TikTok US（獨立公司）15% 股權及董事會席位。將以權益法認列，自 Q4 FY26 起產生非營業收入。這不會影響營收（Oracle 仍為技術供應商），但會貢獻淨利。具體金額尚未披露，需觀察 Q4 表現。</p>
</div>

## 6. ⚠️ 風險提示

<h3 class="risk-red">🔴 高風險</h3>
<div class="sb rh">
<ul>
<li><b>CapEx 超級週期的回報不確定性</b>：TTM CapEx $482 億 vs OCF $235 億，建設速度遠超現金創造能力。若 AI 需求放緩或大型合約延遲，可能導致資產減值或融資壓力。RPO $5,530 億雖高，但取決於資料中心完工進度</li>
<li><b>毛利率結構性下降</b>：從 FY21 的 80.6% 持續下滑至 Q3 FY26 的 64.6%。IaaS 佔比提升是主因，但若 SaaS 增速持續放緩（+11%），整體毛利率可能進一步承壓</li>
<li><b>債務槓桿快速上升</b>：TTM 總債務 $1,622 億 vs FY25 $1,090 億，淨負債 $1,230 億。利率環境若惡化，利息支出將進一步壓縮利潤</li>
</ul>
</div>

<h3 class="risk-yellow">🟡 中風險</h3>
<div class="sb rm">
<ul>
<li><b>AI 基礎設施需求的可持續性</b>：AI 基礎設施收入 +243%、RPO +325% 高度依賴少數超大合約（如 Stargate）。若科技巨頭 AI 支出放緩，Oracle 將首當其衝</li>
<li><b>SaaS 增速放緩</b>：Cloud Applications +11%（CC）低於整體雲端 +44%，Fusion CX 僅 +6%。Oracle 需要證明 AI Agent 和新 CX 產品能重新加速 SaaS 增長</li>
<li><b>裁員影響</b>：高達 18% 裁員雖降低營運成本，但可能短期影響執行力和人才流失</li>
<li><b>客戶集中度</b>：AI 基礎設施收入高度集中於少數超大規模客戶（Stargate/Azure/Meta 等），單一客戶流失影響巨大</li>
</ul>
</div>

<h3 class="risk-green">🟢 機會</h3>
<div class="sb rl">
<ul>
<li><b>MultiCloud DB 是隱藏催化劑</b>：+531% YoY 的增速可能重塑資料庫市場。Oracle DB 直接嵌入 AWS/Azure/GCP 打破了客戶「必須選邊站」的限制</li>
<li><b>垂直 SaaS 深化</b>：Industry SaaS +19%，AI Agent 已嵌入 1,000+ 應用場景。Oracle 在醫療、金融、飯店等垂直市場的壟斷地位持續鞏固</li>
<li><b>TikTok US 投資</b>：15% 股權可能帶來顯著非營業收入，且強化 Oracle 在社交/內容領域的戰略佈局</li>
<li><b>供需缺口縮小</b>：製造據點 3 倍、機櫃產出 4 倍，資料中心建設瓶頸逐步解除，有望加速 RPO 轉化為營收</li>
</ul>

## 8. 📋 關鍵觀察

<ul>
<li>🏆 <b>15 年里程碑</b>：Q3 FY2026 是 Oracle 自 2009 年以來首次有機總營收與 Non-GAAP EPS 同時 YoY +20%+，標誌著從「緩慢成長的老牌軟體公司」向「AI 雲端巨獸」的轉型正在兌現</li>
<li>🚀 <b>OCI +84%、AI 基礎設施 +243%、MultiCloud DB +531%</b>：這三個增速數據代表 Oracle 正在建立新的成長引擎，而不僅僅依靠傳統資料庫和 SaaS</li>
<li>💰 <b>RPO $5,530 億 = 8.6x 年營收</b>：這是 Oracle 歷史上最高的「未來營收安全墊」，但轉化速度取決於資料中心建設進度，且高度集中於少數超大合約</li>
<li>⚠️ <b>CapEx $482 億 vs OCF $235 億</b>：建設超級週期的最大風險在於回報不確定性。$500 億融資計畫和合作夥伴出資模式是緩解因素，但槓桿率已快速上升</li>
<li>🔄 <b>毛利率從 80% 降至 65% 是「選擇」而非「問題」</b>：Oracle 選擇以低毛利但高增速的 IaaS 來換取規模，營業利益率反而提升。這是正確的戰略選擇，但市場需要時間接受這個新常態</li>
</ul>
