## 隱藏的魔鬼細節

<div class="sb">
<h3>🎯 洞察一：Azure +30% 成長 — AI 推動雲端加速</h3>
<p>Azure 雲端營收 Q2 FY26 YoY +30%，其中約 16 個百分點來自 AI 工作負載：</p>
<ul>
<li>這是 Azure 連續第 5 個季度加速成長（從 +26% → +29% → +31% → +33% → +30%）</li>
<li>AI 相關營收年化 run rate 約 $130-150億</li>
<li>Copilot（企業 AI 助手）訂閱用戶持續增長，ASP 高（$30/用戶/月）</li>
</ul>
</div>

<div class="sb">
<h3>🎯 洞察二：營業利益率 47% — 軟體業最強</h3>
<p>Microsoft 的營業利益率 47% 在 Mag 7 中最高：</p>
<ul>
<li>高於 Apple（~33%）、Google（~32%）、Meta（~41%）</li>
<li>來源：<b>高毛利軟體</b>（Windows、Office、Azure）+ <b>低邊際成本</b></li>
<li>Azure 的邊際利潤率隨規模持續擴張</li>
</ul>
</div>

<div class="sb">
<h3>🎯 洞察三：CapEx 暴增但 FCF 仍強</h3>
<ul>
<li>Microsoft FY2026 CapEx 預計達 $800-900億（vs FY2025 ~$560億）</li>
<li>但 TTM FCF 仍達 $774億 — 因為 Azure 的高毛利和 Office 訂閱的經常性收入能吸收 CapEx 衝擊</li>
<li>這是 Mag 7 中「CapEx 最多但 FCF 最穩」的公司</li>
</ul>
</div>

<div class="sb">
<h3>🎯 洞察四：OpenAI 投資 — 最聰明的 AI 佈局</h3>
<ul>
<li>Microsoft 對 OpenAI 投資約 $130億（累計）</li>
<li>獨家 Azure 雲端合作 + GPT 模型整合至 Copilot/Office/Azure</li>
<li>這不是「被 OpenAI 綁架」而是「用 OpenAI 鎖定企業 AI 市場」</li>
<li>OpenAI 營收增長直接轉化為 Azure 的工作負載增長</li>
</ul>
</div>

<li>Copilot 企業訂閱 ASP 高、黏性強</li>
<li>轉換成本極高 — 企業客戶幾乎不可能離開 Microsoft 生態</li>

## 總結

- 🟢 Q2 FY26 營收 **$813億**（+16.7%），營業利益率 **47%**
- 🟢 Azure +30%，AI 工作負載佔比持續提升
- 🟢 TTM 淨利首破 **$1,000億**，EPS **$15.99**
- 🟢 Copilot + OpenAI = 企業 AI 最強組合
- 🟡 CapEx $800-900億，AI 回報需時間驗證

**一句話**：Microsoft 是 Mag 7 中「最穩的 AI 賭注」— 47% 的營業利益率證明它不需要賭博就能贏，Azure + Copilot + OpenAI 的三重 AI 佈局讓它在企業 AI 市場幾乎無法被挑戰。
